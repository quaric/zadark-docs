# Cài đặt ZaDark cho Zalo Web

Kích hoạt Dark Mode (Chế độ tối) cho Zalo Web trên Chrome, Safari, Edge và Firefox.

:::info
Bằng cách chấp nhận cài đặt và sử dụng ZaDark, bạn **ĐỒNG Ý** với [**Điều khoản sử dụng**](/privacy-policy) của chúng tôi.
:::

<CardsInstallWeb />

:::tip
Trình duyệt sẽ tự động cập nhật ZaDark khi có phiên bản mới.
:::

:::danger
Chúng tôi ngừng phát hành bản cập nhật **ZaDark for Opera** từ ngày 04/01/2024. Bạn có thể cài đặt [**ZaDark for Chrome**](https://chrome.google.com/webstore/detail/zadark-%E2%80%93-zalo-dark-mode/llfhpkkeljlgnjgkholeppfnepmjppob) để thay thế.
:::
